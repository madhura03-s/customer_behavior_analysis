📊 Customer Behavior Analysis

Data analytics project analyzing customer behavior using Python (Pandas), SQL, and Power BI. Includes data cleaning, transformation, SQL analysis, and dashboard creation to uncover insights on customer spending and segmentation.

🔹 Overview

This project demonstrates an end-to-end data analytics workflow, from raw CSV data to actionable insights. It focuses on cleaning, transforming, and analyzing customer purchase data to understand behavior patterns.

📁 Dataset

CSV dataset containing customer details such as Customer ID, Gender, Age, Category, Purchase Amount, Review Rating, Discount Usage, and Purchase Frequency.
Used for analysis and learning purposes.

🛠️ Tools
Python (Pandas, SQLAlchemy)
SQL (MySQL)
Power BI

⚙️ Steps
Loaded dataset using Pandas
Performed data exploration (info(), describe(), null checks)
Handled missing values using median (grouped by category)
Cleaned column names (lowercase, removed spaces/symbols)
Created new features:
Age groups using quartiles
Purchase frequency in days
Validated columns and removed redundant data
Loaded cleaned data into MySQL using SQLAlchemy
Performed SQL queries for analysis
Built interactive dashboard in Power BI
Created report and presentation
📊 Dashboard

📈 Results
Identified customer spending patterns across categories
Analyzed relationship between discounts and purchases
Segmented customers based on age groups
Standardized purchase frequency for better analysis
